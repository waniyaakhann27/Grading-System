# Student Certification Grading System

A super simple Python tool that helps teachers quickly check if students have submitted all required certificates and the assignment — no manual counting, no mistakes.

### What it checks (8 items = 8 points)
1. Vidyarthi certificate  
2. Great Learning certificate  
3. HackerRank 5-star proof  
4–7. Four extra certificates  
   - Python Basic  
   - Problem Solving Basic  
   - Problem Solving Intermediate  
   - Python Foundation Test  
8. 35 Codes Assignment  

Each "yes" = 1 point → Final score out of 8 + Grade (A+, A, B+, B, C or Incomplete)

### Features
- Clean, easy-to-read output  
- Optional student name  
- Just answer yes/no (or y/n)  
- One single file — no installation needed  

### How to Use
1. Make sure Python is installed  
2. Download `grading_system.py`  
3. Double-click or run in terminal:  
   ```bash
   python grading_system.py
